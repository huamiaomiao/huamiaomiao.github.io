## 使用方式

## 核心技术基础

### HTML5 基础

- [ ] 语义化标签与可访问性（a11y）
- [ ] 表单与交互元素（校验、约束、可用性）
- [ ] 多媒体与画布：Audio/Video、Canvas、SVG
- [ ] 存储与离线：Storage、IndexedDB、Cache

### CSS3 进阶

- [ ] 选择器与优先级、继承与层叠
- [ ] 盒模型、BFC、浮动与清除
- [ ] 现代布局：Flexbox、Grid
- [ ] 动画与过渡、Transforms
- [ ] 响应式与移动端适配、媒体查询
- [ ] 预处理/后处理：Sass/Less、PostCSS
- [ ] CSS 架构与规范：BEM / ITCSS

### JavaScript 核心

- [ ] 语言基础与 ES6+（解构、类、模块、迭代器、Proxy）
- [ ] 原型链、作用域、闭包、this 绑定
- [ ] 模块化方案：ESM / CommonJS
- [ ] DOM/BOM、事件模型与委托
- [ ] 异步编程：Promise / async-await
- [ ] 事件循环与任务队列（宏/微任务）
- [ ] 设计模式与编码范式（FP/OOP）
- [ ] 数据结构与算法基础（时间/空间复杂度）
