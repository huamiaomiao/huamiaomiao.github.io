## 思维导图：前端开发工程师学习路径

- 前端开发工程师学习路径

  - 核心技术基础

    - HTML5 基础

      - 语义与文档
        - 必备：语义化标签 `header/nav/main/section/article/aside/footer`
        - 必备：标题层级与文档结构、可读面包屑与站点结构
        - 必备：可访问性基础（A11y）：语义/可聚焦/键盘导航/ARIA 最小集
        - 必备：SEO 基础：语义结构、标题/摘要、描述性链接/图片替代文本
        - 细分知识点：
          - 块级/内联语义元素的选择与替代方案
          - 标题大纲与层级（H1-H6）规范、跳过层级的影响
          - ARIA 常用 role（banner/navigation/main/contentinfo/alert/dialog）
          - 焦点顺序与 tabindex、:focus-visible 与焦点样式
          - 元信息（meta/OG）与结构清晰的文案
      - 表单与交互基础
        - 必备：`input/textarea/select` 常用类型与属性、`label` 关联
        - 必备：原生校验 required/pattern/min/max/step 与错误提示
        - 必备：表单可用性与无障碍（占位/提示/错误反馈/聚焦顺序）
        - 细分知识点：
          - 常用 input 类型（text/email/url/number/date/file/color 等）
          - Constraint Validation API 与原生校验触发机制
          - FormData/多文件上传、禁用/只读/必填的差异
          - 渐进增强：无 JS 也可提交与回退策略
      - 布局与响应式
        - 必备：移动优先、断点设计、流式布局
        - 必备：视口与像素：`<meta viewport>`、DPR、CSS 像素
        - 细分知识点：
          - 断点选择策略（内容驱动 vs 设备驱动）、常见断点组合
          - 弹性单位：%/vw/vh/vmin/vmax、排版单位：rem/em/ch
          - 栅格与等比缩放、容器宽度与最大宽度限制
      - 多媒体与 Canvas
        - 必备：`audio/video` 基础、控制属性与事件
        - 细分知识点：
          - 控制属性：controls/autoplay/muted/playsinline/preload
          - 媒体事件与自定义控件、字幕与可访问性

    - CSS3 进阶

      - Flexbox 布局
        - 必备：主轴/交叉轴、容器与项目属性（`justify-content/align-items/flex`）
        - 必备：常见布局模式：两栏/三栏、自适应侧边栏、水平垂直居中
        - 细分知识点：
          - 容器属性：flex-direction/wrap/flow/gap/justify/align/align-content
          - 项目属性：order/flex-grow/flex-shrink/flex-basis/align-self
          - flex 简写与典型布局（圣杯/双飞翼/居中/等高列）
      - Grid 布局
        - 必备：行列轨道、区域命名、`minmax/auto-fit/auto-fill`
        - 细分知识点：
          - grid-template、命名区域、repeat/fr/minmax
          - 自动放置、隐式网格、对齐与间距
      - 动画与过渡
        - 必备：过渡 `transition`、关键帧 `@keyframes/animation`
        - 细分知识点：
          - 过渡属性/时长/延迟/缓动函数（cubic-bezier/steps）
          - 动画填充模式、循环与方向、动效可访问性（降低动画）
      - 预处理器（Sass/Less）
        - 必备：变量/嵌套/Mixin/函数/继承，占位选择器与分片组织
        - 细分知识点：
          - 局部化与分层、色板/间距/字号的系统化
          - 常见工程集成与构建产物控制
      - 媒体查询与响应式策略
        - 必备：媒体特性（宽高/分辨率/首选配色/触控）
        - 细分知识点：
          - min/max-width、resolution、prefers-color-scheme/motion
          - pointer/hover、light/dark 主题切换
      - 规范与工程建议
        - 必备：选择器优先级管理、BEM 命名
        - 细分知识点：
          - 特指度与层叠、作用域控制、可维护样式结构

    - JavaScript 核心（严格对齐现有目录）
      - 入门基础
        - 必备：ES5 与 ES6 的定位与差异概览
        - 必备：严格模式（`'use strict'`）的行为变化与影响
        - 必备：ES5 常用扩展与实战用法
        - 细分知识点：
          - 概览：语法新增、标准推进、向后兼容与转译
          - 严格模式：禁止隐式全局/重复参数/with，this 非自动装箱
          - ES5 扩展：Object.create/defineProperty/keys，Array 的 map/filter/reduce/some/every/indexOf，Function.prototype.bind
      - ES6 语法基石
        - 变量：`let/const` 与块级作用域、TDZ、不可重复声明
        - 解构赋值：数组/对象/默认值/嵌套与常见坑
        - 箭头函数：词法 this、参数与返回、与普通函数差异
        - 剩余参数与展开：`...rest`/`...spread` 在函数/数组/对象中的使用
        - 字符串/数组/对象扩展：模板字符串、includes/find/findIndex、Object.assign/entries/values 等
        - Set 数据结构：去重/集合运算、与 Map/Weak\* 的区别（概念对比）
        - 细分知识点：
          - let/const 与闭包循环变量、暂时性死区错误排查
          - 箭头函数：不绑定 this/arguments/new，适用与不适用场景
          - 展开/剩余：不可变更新、浅拷贝特性与注意事项
      - 异步与模块化
        - Promise 与异步编程：状态机、链式调用、错误传播、组合（all/race/allSettled）
        - async/await（在 Promise 基础上的语法）：顺序/并发、错误捕获
        - 模块化 import/export：命名与默认导出、循环依赖与树摇概念
        - 细分知识点：
          - 微任务与宏任务的调度差异、常见异步题型
          - all 与 race 的使用策略、超时与取消（AbortController 概念）
      - 面向对象
        - 类与面向对象：构造器、实例/静态成员、继承/`super`
        - 原型与原型链：属性查找规则、`instanceof/hasOwnProperty`
        - 细分知识点：
          - 组合优于继承的思考、私有字段与封装（概念）
      - 实战练习与结果输出（与现有练习对齐）
        - 练习：作用域/解构/箭头混合题（含输出）
          - 步骤：阅读题目 → 手动推导输出与原因 → 运行验证 → 总结规律
        - 实战：原生 JS 实现 Todo 清单
          - 步骤：结构与样式 → 任务增删改查 → 本地存储 → 交互与可访问性
        - 实战：Fetch 请求 + DOM 渲染
          - 步骤：拉取数据 → 状态与错误处理 → DOM 绑定与更新 → 空态与加载态

  - 主流框架与库

    - Vue 基础（从 0-1）
      - 入门与环境
        - 必备：基础概念、模板语法、响应式数据模型
        - 必备：系统指令基础 `v-bind/v-model/v-on/v-if/v-for`
        - 必备：事件修饰符 `.stop/.prevent/.capture` 与键盘修饰符
        - 进阶：条件/列表渲染优化、`key` 使用、动态组件
        - 细分知识点：
          - 模板表达式与指令语法、计算/侦听的边界
          - v-model 修饰符与表单控件配套
          - 组件异步加载与按需渲染
      - 组件与通信
        - 必备：组件定义与注册（全局/局部）、SFC 基本结构
        - 必备：`props/emit`、父子通信、`v-model` 组件
        - 进阶：`provide/inject`、插槽（默认/具名/作用域插槽）
        - 必备：生命周期（创建/挂载/更新/卸载）与常见时机
        - 细分知识点：
          - 受控/非受控组件、单向数据流
          - 作用域插槽数据传递与组合
          - 生命周期与 DOM/网络请求时机
      - 路由与状态
        - 必备：路由表、动态/嵌套路由、懒加载与代码分割
        - 进阶：导航守卫（全局/路由/组件）、权限与重定向、滚动行为
        - 必备：Pinia：Store 设计、State/Getters/Actions、模块化
        - 细分知识点：
          - 动态参数与别名、404 与重定向、路由元信息
          - Store 切片与依赖隔离、组合式 Store
      - 数据与异步
        - 必备：HTTP 请求（fetch/Axios）、拦截器与错误处理
        - 必备：计算属性与侦听器（副作用隔离与性能）
        - 细分知识点：
          - 取消请求、重试机制、统一错误提示
          - watch/watchEffect 差异与清理
      - 进阶特性
        - 进阶：自定义指令与按键修饰符扩展、格式化工具迁移策略
        - 进阶：动画与过渡：`<transition>/<transition-group>`、路由过渡
        - 必备：Composition API：`ref/reactive/computed/watch`、可复用 hooks
        - 细分知识点：
          - 组合式函数抽象、依赖追踪与内存清理
          - 过渡状态与路由切换动效注意事项
      - 实战练习与结果输出（可运行）
        - 列表渲染/过滤/排序/分页、表单双向绑定与校验
        - Todo 组件化设计、状态提升与跨组件通信
        - 路由与动态详情、嵌套路由与守卫
        - Pinia 计数器/异步请求/持久化

  - 工具链与工程化
    - 构建工具
      - 必备：Vite 开发服务器/HMR、依赖预构建、产物拆分与性能
      - 细分知识点：
        - Vite 模式与环境变量、别名与代理、静态资源与拆包
        - 产物分析与缓存策略、Sourcemap 与错误定位
