# 组件的定义与注册

> 目标：完成从“定义 → 注册 → 使用”的完整闭环，理解局部与全局注册差异。

## 定义组件（单文件组件 SFC）

```vue
<!-- src/components/Hello.vue -->
<template>
  <h2>Hello {{ name }}</h2>
</template>
<script>
export default { props: { name: { type: String, default: "Vue" } } };
</script>
```

## 局部注册（推荐）

```vue
<template><Hello name="Ada" /></template>
<script>
import Hello from "./components/Hello.vue";
export default { components: { Hello } };
</script>
```

## 全局注册（插件/入口）

```js
import Hello from "./components/Hello.vue";
const app = createApp(App);
app.component("Hello", Hello);
```

## 命名规范

- 组件名使用 PascalCase 或 kebab-case，一致且语义清晰
- 目录建议：`base/`（基础）、`shared/`（通用）、`features/`（功能）

## 插槽与作用域插槽（预告）

```html
<Card>
  <template #header>标题</template>
  内容
  <template #footer="{ count }">共 {{ count }} 条</template>
</Card>
```

## 练习建议

- 创建 `Card`、`Button`、`Empty` 基础组件并全局注册
- 将列表项抽成 `TodoItem` 局部注册使用
