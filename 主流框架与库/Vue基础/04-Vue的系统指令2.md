# Vue 的系统指令 2（进阶）

> 目标：补充更实用的指令用法，完善日常开发能力。

## v-bind 简写与动态参数

```html
<a :href="url">跳转</a>
<div :[attrName]="value"></div>
<!-- 动态属性名 -->
```

## v-model 的修饰符

```html
<input v-model.lazy="name" />
<!-- change 时同步 -->
<input v-model.number="age" />
<!-- 自动转 number -->
<input v-model.trim="keyword" />
<!-- 去除首尾空格 -->
```

## v-on 的动态事件名

```html
<button @[eventName]="handler">动态事件</button>
```

## v-slot（插槽语法糖）

```html
<card>
  <template #header>标题</template>
  内容
  <template #footer>底部</template>
</card>
```

## v-once / v-memo（性能）

```html
<p v-once>{{ heavyRenderText }}</p>
<!-- 仅渲染一次 -->
```

## v-cloak（防闪烁）

与样式配合，避免初次渲染的 Mustache 闪现：

```html
<style>
  [v-cloak] {
    display: none;
  }
</style>
<div id="app" v-cloak>{{ msg }}</div>
```
