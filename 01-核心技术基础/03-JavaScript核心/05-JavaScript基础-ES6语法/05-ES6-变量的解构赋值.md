# ES6：变量的解构赋值

> 目标：用更简洁直观的方式从数组/对象中取值。

## 数组解构

```js
const [a, b = 0, ...rest] = [1, undefined, 3, 4];
// a=1, b=0, rest=[3,4]
```

## 对象解构与重命名

```js
const { id, name: displayName = "Guest" } = { id: 1 };
// id=1, displayName="Guest"
```

## 函数参数解构

```js
function draw({ x = 0, y = 0, color = "black" } = {}) {
  /* ... */
}
```

## 嵌套与默认值

```js
const user = { profile: { city: "SH" } };
const { profile: { city = "BJ" } = {} } = user;
```
