# ES5 中的一些扩展（精选）

> 目标：掌握在 ES5 引入的常用 API，便于读写老代码与过渡。

## 数组方法

```js
[1, 2, 3].forEach((x) => console.log(x));
[1, 2, 3].map((x) => x * 2);
[1, 2, 3].filter((x) => x > 1);
[1, 2, 3].some((x) => x === 2);
[1, 2, 3].every((x) => x > 0);
[1, 2, 3].reduce((a, b) => a + b, 0);
Array.isArray([]); // true
```

## Function.prototype.bind

```js
const obj = { x: 1 };
function print() {
  console.log(this.x);
}
const bound = print.bind(obj);
bound(); // 1
```

## Object API

```js
Object.create(null); // 无原型对象
Object.keys({ a: 1, b: 2 }); // ["a","b"]
Object.defineProperty(obj, "x", { value: 1, writable: false });
```

## JSON

```js
JSON.stringify({ a: 1 });
JSON.parse('{"a":1}');
```
