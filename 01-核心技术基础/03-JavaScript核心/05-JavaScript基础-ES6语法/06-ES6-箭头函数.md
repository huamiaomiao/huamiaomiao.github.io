# ES6：箭头函数

> 目标：理解简洁语法与 `this` 绑定特性，避免常见误用。

## 语法

```js
const add = (a, b) => a + b;
const toUpper = (s) => s.toUpperCase();
const make = () => ({ ok: true }); // 返回对象需括号包裹
```

## this 绑定

- 箭头函数不绑定 `this`，会捕获外层 `this`
- 不能用作构造器（不可 `new`）
- 无 `arguments`（使用剩余参数）

```js
const vm = {
  count: 0,
  incLater() {
    setTimeout(() => {
      this.count++;
    }, 0); // this 指向 vm
  },
};
```

## 何时不要用箭头

- 需要 `this` 可变/动态绑定
- 需要 `new` 构造实例
- 需要 `arguments` 的场景（可改用剩余参数）

## 练习

- 将回调中丢失 `this` 的写法改成箭头函数捕获外层 `this`
