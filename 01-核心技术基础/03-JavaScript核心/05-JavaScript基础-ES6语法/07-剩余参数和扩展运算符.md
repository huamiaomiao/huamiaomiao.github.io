# 剩余参数和扩展运算符（...）

> 目标：掌握函数的可变参数与数组/对象的展开与拷贝。

## 剩余参数

```js
function sum(...nums) {
  return nums.reduce((a, b) => a + b, 0);
}
```

## 数组展开

```js
const a = [1, 2],
  b = [3];
const c = [...a, ...b]; // [1,2,3]
const copy = [...a]; // 浅拷贝
```

## 对象展开

```js
const user = { id: 1, name: "Ada" };
const withAge = { ...user, age: 20 }; // 浅合并
```

注意：展开是浅拷贝，嵌套对象引用不变。

## 练习

- 用剩余参数封装 `max(...nums)`
- 用对象展开合并默认配置与用户配置
