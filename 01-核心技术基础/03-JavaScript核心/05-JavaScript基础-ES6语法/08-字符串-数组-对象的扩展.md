# 字符串、数组、对象的扩展（ES6+ 精选）

> 目标：掌握最常用的新增方法与语法糖。

## 字符串扩展

```js
"hello".startsWith("he");
"hello".endsWith("lo");
"hello".includes("ell");
"x".repeat(3); // "xxx"
```

## 数组扩展

```js
Array.from("abc"); // ["a","b","c"]
Array.of(1, 2, 3); // [1,2,3]
[1, 2, 3].find((x) => x > 1); // 2
[1, 2, 3].findIndex((x) => x === 3); // 2
[1, 2, 3].includes(2); // true
```

## 对象扩展

```js
Object.assign({}, { a: 1 }, { b: 2 });
Object.entries({ a: 1 }); // [["a",1]]
Object.fromEntries([["a", 1]]); // {a:1}
const { a, ...rest } = { a: 1, b: 2, c: 3 }; // 剩余属性
```

## 模板字符串与标签模板

```js
const name = "Ada";
const s = `Hello ${name}!`;
```

## 练习

- 用 `Object.fromEntries` 将 `[['k',1],['v',2]]` 转为对象
- 用 `find`/`includes` 重写老代码中的下标判断
